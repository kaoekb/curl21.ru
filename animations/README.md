# Additional Animations

Keep the main `curl21.ru` animation in `frames/`.

To add a new animation route like `curl21.ru/duck`, create a new directory:

```text
animations/
  duck/
    0.txt
    1.txt
    2.txt
```

Each frame should:
- use the `.txt` extension
- keep the same width and height across all files
- be named in numeric order
